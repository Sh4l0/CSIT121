CSS Rules | Entry Two

After learing CSS, making websites has become a lot more fun. This can be done through creating rules, which are made up of selectors, propeties, and values.

Best Design Practices| Entry Three

In this unit, we learned about responsive layouts and the modern standards for web design

Key Takeaways

  1. Use media queries to allow your site to adapt to the device its on
  2. When using headings, ensure that they appear in decending order (H1->H6)
  3. Utilizing relative units (like % or em) to allow universal formatting
